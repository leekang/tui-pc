### Col

布局列

#### 何时使用

需要栅格布局

#### API

| 参数							| 说明							| 类型					| 默认值|
| -------------			|:-----------------:| -------------:| -----:|
| className					| 类名							|	String				| 无		|
| span							| 默认宽度屏幕的栅格| Number				| 无		|
| sm								| 小屏幕的栅格			| Number				| 无		|
| span							| 大屏幕的栅格			| Number				| 无		|


栅格数24
也可以直接使用style或者class直接调整百分比，实现24栅格外的布局

尺寸信息
sm	:320px  iphone5
span:375px	iphone6
lg	:414px	iphone6+

具体数值可以在theme.scss中配置layout相关变量
