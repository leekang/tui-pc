###splitter
分割线
###如何使用
            <Splitter width={560} height={1} background={222} type="dash"/>
###API
参数              描述          可选值
width            分割线的宽度    数值，默认auto
height           分割线的高度    数值，默认1
style             分割线的类型    dash(虚线),normal(正常实线),special(特殊分割线)
background       分割线的颜色    颜色值如222,d9d9d9
