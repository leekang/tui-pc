### Modal

弹窗

#### 何时使用

需要用户确定信息时

#### API

| 参数							| 说明							| 类型					| 默认值|
| -------------			|:-----------------:| -------------:| -----:|
| className					| 类名							|	String				| 无		|
| wrapClassName			| 窗口部分类名			|	String				| 无		|
| title							| 标题							| React.Element	| 无		|
| footer						| 尾部视图					| React.Element	| 无		|
| onOk							| 确认事件					| Function			| 无		|
| onCancel					| 取消事件					| Function			| 无		|
| visible						| 是否显示					| Boolean				| false	|
| maskClosable			| 点击背景关闭			| Boolean				| true	|
| closable					| 关闭符号显示			| Boolean				| false	|
| width							| 弹窗宽度					| Number、String| 70%		|
| okText						| 确认文案					| String				| 确定	|
| cancelText				| 取消文案					| String				| 取消	|
