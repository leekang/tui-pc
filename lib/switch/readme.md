### Switch

切换组件

#### 何时使用

改版布尔类型的变量值

#### API

| 参数							| 说明							| 类型					| 默认值|
| -------------			|:-----------------:| -------------:| -----:|
| className					| 类名							|	String				| 无		|
| name							| form name					| String				| 无		|
| defaultChecked		| 默认选中					| Boolean				| 无		|
| checked						| 选中状态					| Boolean				| 无		|
| disabled					| 不可修改					| Boolean				| 无		|
| size							| 尺寸							| String				| true	|
