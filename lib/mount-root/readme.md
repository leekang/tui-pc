### Mount-root

根节点挂载

#### 何时使用

需要将视图节点挂载到根节点，并且关闭后去除节点

#### API

| 参数							| 说明							| 类型					| 默认值|
| -------------			|:-----------------:| -------------:| -----:|
| visible						| 是否显示					| Boolean				| false	|
| closeDelay				| 窗口关闭延迟毫秒	| Number				| 0			|
| getContainer			| 设置挂载父节点		| Function(node)| 无		|
